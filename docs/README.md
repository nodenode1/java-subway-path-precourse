## 기능목록


### Domain
####역
 - 거리 정보 저장
 - 시간 정보 저장
####라인
 - 라인 별 역 저장 
 - 최단 거리 계산
 - 최소 시간 계산

### View
 - InputView
 - OutputView

### Controller
 - 메인화면 기능 선택
 - 최단 거리 경로 선택
 - 최소 시간 경로 선택
 - 조회 경로 출력
 - 경로 기준 화면 선택
### Exception
 - 출발역과 도착역이 같으면 에러 처리
 - 출박역과 도착역이 이어지지 않았다면 에러 처리
 - 존재하지 않는 역 사용시 에러 처리
 - 기능 선택 시 입력 값이 메뉴 네임이 아닌경우 에러처리
